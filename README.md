# 🎬 ReelFeel

Plataforma de curadoria de filmes com catálogo estilo Netflix, gamificação e comunidade cinéfila.

## Começar

```bash
open index.html
```

## Páginas

| Página | Descrição |
|--------|-----------|
| **Home** | Catálogo com hero carousel, filtros "Surpreenda-me", seções temáticas |
| **Feed** | Avaliações da comunidade, resenhas em vídeo, posts com likes/comentários |
| **Minha Curadoria** | Perfil do usuário, estatísticas, filmes assistidos, resenhas |

## Funcionalidades

- **3 Perfis**: Amanda, Ângelo, Visitante (experiências personalizadas)
- **Gamificação**: XP por ações, 7 níveis, badges de perfil
- **Surpreenda-me**: Filtros por mood, gênero e XP
- **Cards interativos**: Play, lista, avaliação rápida
